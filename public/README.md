# Public Directory

This directory contains static assets for your website.

## Files to Add

1. **resume.pdf** - Place your resume PDF here. The download link on the homepage will automatically work.

2. **images/** - Place all project images in this directory:
   - Use descriptive filenames
   - Recommended formats: JPG, PNG, WebP
   - Recommended size: 800x600px or similar aspect ratio
   - Reference them in your project data as `/images/your-image.jpg`

## Example Structure

```
public/
├── resume.pdf
└── images/
    ├── project-1.jpg
    ├── project-2.png
    └── my-app.webp
```
