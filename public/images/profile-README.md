# Profile Images

Place your profile pictures here. The carousel will automatically display all images you add.

## Instructions

1. Add your profile images to this directory
2. Name them: `profile-1.jpg`, `profile-2.jpg`, `profile-3.jpg`, etc.
3. Update the image paths in `/app/page.tsx` in the ProfileCarousel component

## Recommended

- Square aspect ratio (1:1) works best
- Recommended size: 600x600px or larger
- Supported formats: JPG, PNG, WebP
- Keep file sizes reasonable for fast loading

## Example

```
/images/
  ├── profile-1.jpg
  ├── profile-2.jpg
  └── profile-3.jpg
```
